# Proyecto Vacunación COVID-19

Aplicación en C# que simula una campaña de vacunación utilizando teoría de conjuntos.

## Requisitos cumplidos

- 500 ciudadanos ficticios
- 75 vacunados con Pfizer
- 75 vacunados con AstraZeneca
- Operaciones de teoría de conjuntos:
  - Unión
  - Intersección
  - Diferencia

## Listados generados

- Ciudadanos no vacunados
- Ciudadanos con ambas dosis
- Ciudadanos solo Pfizer
- Ciudadanos solo AstraZeneca

## Ejecución

```bash
dotnet run
```
