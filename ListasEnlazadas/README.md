
# Listas Enlazadas en C#

Este proyecto contiene la implementación de los siguientes ejercicios:

## Ejercicio 1
Función que calcula el número de elementos de una lista enlazada.

## Ejercicio 3
Método de búsqueda que retorna cuántas veces aparece un dato dentro de la lista enlazada.

El proyecto está desarrollado en **C#**, utilizando listas enlazadas implementadas manualmente mediante nodos.
